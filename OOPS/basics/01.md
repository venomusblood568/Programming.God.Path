# classes and object

```
class HelloWorld {                // Class
    public static void main(String[] args) {
       person p1 = new person();  // Object
       p1.name = "sam";           // Setting object properties
       p1.age = 21;               // Setting object properties
       System.out.println(p1.name);
       System.out.println(p1.age);
    }
}

class person {                    // Class
    String name;                  // Attribute of the class
    int age;                      // Attribute of the class
}

```
